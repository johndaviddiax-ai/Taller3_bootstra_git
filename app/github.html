    1. Estructura del proyecto 

    

    Estructura de las carpetas del proyecto:
    
taller3/

│── app/
│    ├── forms.html
│    ├── tables.html
│    └── github.html
│── assets/
│    ├── css/
│    ├── img/
│    └── js/
│── index.html
│── README.md

    2. Requisitos iniciales

    • Tener cuenta activa en GitHub
    • Tener Git instalado
    • Tener Visual Studio Code
    • Abrir el proyecto desde VS Code

    3. Configuración de Git

    Configuración del usuario en Git:

    git config --global user.name "Tu Nombre"

    git config --global user.email "correo@ejemplo.com"


    Verificar configuración:

    git config --global --list

    4. Inicializar Git en el proyecto

    Ubicarse en la carpeta del proyecto y ejecutar:

    git init

    git add .

    git commit -m "Primer commit: estructura inicial"

    Esto permite crear un historial de cambios del proyecto.

    5. Crear repositorio en GitHub

    1. Ingresar a GitHub
    2. Hacer clic en "New repository"
    3. Agregar un nombre al repositorio, ej: taller3_bootstrap_github
    4. Seleccionar si sera publico o si sera privado
    5. Crear el repositorio sin agregar: README si el proyecto ya tiene archivos en local

    6. Conectar proyecto local con Github

    Copiar la URL del repositorio creado y ejecutar:

    git remote add origin https://github.com/usuario/proyecto-bootstrap-adso.git

    git branch -M main

    git push -u origin main

    Cambia usuario y proyecto por datos reales de tu reposotorio

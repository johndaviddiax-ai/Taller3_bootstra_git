# Taller 3 - Bootstrap 

Sitio web desarrollado con HTML y Bootstrap 5 como práctica de maquetación y diseño web responsivo.

 **Sitio en vivo:** [Ver página](https://johndaviddiax-ai.github.io/Taller3_bootstra_git/)

---

 Páginas

 Inicio (`index.html`)
Página principal compuesta por:
- 5 secciones de contenido
- Carrusel de imágenes interactivo
- Barra de navegación con acceso a todas las secciones

 Tablas (`app/tables.html`)
Tabla con información de los países del mundo:
- 10 columnas x 20 filas
- Diseño responsivo con Bootstrap 5

###  GitHub (`app/github.html`)
Presentación del proceso utilizado para subir el proyecto a GitHub, documentando cada paso del flujo de trabajo con Git.

###  Acceso (`app/forms.html`)
Formulario para que los usuarios puedan:
- Registrarse
- Iniciar sesión

---

##  Tecnologías

- HTML5
- Bootstrap 5
- Bootstrap Icons

---

##  Cómo usar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/johndaviddiax-ai/Taller3_bootstra_git.git
   ```
2. Y abre el `index.html` en tu navegador preferido
